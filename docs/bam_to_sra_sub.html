NAME

bam_to_sra_sub.pl - Generate SRA submission.

SYNOPSIS

bam_to_sra_sub.pl [options] [file(s)...]

  Required parameters:
    -outdir    -o   Folder to output result to.
    -study     -s   Sample name to be applied to output file.

  Optional:
    -type      -t   Only required if not encoded in readgroup LB tag.
                    Sequencing type, controlled values:
                      WGS     - Whole Genome Seq
                      WXS     - Whole eXome Seq
                      RNA-Seq - RNA seq

  Other:
    -help      -h   Brief help message.
    -man       -m   Full documentation.

  File list can be full file names, wildcards or combination, e.g.
    bam_to_sra_sub.pl -s PANCAN -o myout input/*.bam input2/sample2.bam

OPTIONS

-outdir

Directory to write output to. All SRA XML files will be written to a 'sample' directory in this area, Associated BAM files and MD5s will be symlinked into this area for easy upload.

Please ensure that you will be able to read from the area with GeneTorrent otherwise you will need to copy the data into an appropriate location (remembering to follow symlinks).

-study

The STUDY_REF that this data is to be assigned to in the repository.

-help

Print a brief help message and exits.

-man

Prints the manual page and exits.
